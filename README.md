# AI 101 - AI 도구 추천 서비스

AI 101은 사용자의 질문을 분석하여
**단순한 질문은 즉시 처리**하고,
**복잡한 질문은 세부 작업으로 분할한 뒤 각 단계에 적합한 AI 도구를 추천하고 활용 방법까지 안내하는**
GenAI 기반 AI 도구 추천 서비스입니다.

---

## 실행 환경 준비

다음과 같은 환경 변수 설정이 필요합니다. ex) .env

```env
OPENAI_API_KEY=your_openai_api_key
GOOGLE_API_KEY=your_google_api_key
GOOGLE_SEARCH_ENGINE_ID=your_google_search_engine_id
```

---

## 실행 방법

GenAI 프로젝트 경로에서 아래 명령어를 실행합니다.

```bash
(venv) python main.py
```

실행 후 채팅 인터페이스를 통해 질문을 입력할 수 있습니다.

---

## 사용 방법

### 3.1 간단한 요구사항

#### 입력 예시

```
현재 날짜로부터 GPT 5가 며칠 전에 나왔는지 계산해줘.
```

#### 동작 방식

- 질문이 단순 계산/정보 요청으로 판단됨
- 별도의 계획 수립 없이 즉시 처리
- 필요한 경우 계산기, 시간, Google 검색 도구를 호출
- ReAct 방식으로 빠르게 최종 답변 생성

---

### 3.2 복잡한 요구사항

#### 입력 예시

```
생성형 AI 발표를 위해서 PPT를 만들어야 해.
```

#### 동작 방식

1. 질문이 **복잡한 요구사항**으로 판단됨
2. 즉시 답변을 생성하지 않고, 먼저 업무 수행을 위한 **세부 계획을 제안**
3. 계획은 보통 2~5개의 서브 작업으로 구성됨
   (예: 자료 조사, 슬라이드 구조 설계, 도구 선정 등)
4. 사용자는 자연어로 다음 중 하나를 선택

- 승인

  ```
  네, 이대로 진행해
  ```

- 수정

  ```
  4번은 빼고 진행해줘
  ```

- 취소

  ```
  취소할게
  ```

#### 승인 또는 수정 시

- 각 세부 작업에 대해 적합한 AI 도구를 탐색
- Vector DB(RAG)를 우선 활용하여 내부 지식 검색
- 정보가 부족한 경우에만 Google 검색을 추가 수행
- 각 단계별로:

  - 어떤 AI 도구를 쓰는지
  - 왜 적합한지
  - 어떻게 활용하면 되는지까지 포함하여 최종 답변 생성

#### 취소 시

- 해당 요청에 대한 처리 즉시 종료
- 추가 실행 없음

---

### 3.4 메모리 사용

#### 입력 예시

```
지금까지의 내용을 요약해줘.
```

#### 메모리 기능 설명

- 한 턴의 대화가 종료될 때마다

  - 사용자의 의도
  - 선호
  - 중요한 결정 사항을 자동으로 추출하여 메모리에 저장

- 이후 대화에서 해당 메모리를 불러와 응답에 반영
